# أكاديمية المعارف - لوحة التحكم الإدارية

## نظرة عامة

تم بناء لوحة تحكم إدارية شاملة لأكاديمية المعارف تتضمن جميع الميزات اللازمة لإدارة الدورات التدريبية والطلاب والمدربين والمراجعات.

## المميزات المنفذة

### 1. **لوحة البيانات الرئيسية (Dashboard)**

- عرض إحصائيات شاملة:
    - إجمالي الدورات
    - عدد الطلاب المسجلين
    - عدد المدربين
    - عدد الالتحاقات النشطة
- توزيع الدورات (أون لاين / أوف لاين)
- مؤشرات الأداء:
    - متوسط التقييمات
    - الإيرادات الشهرية
    - معدل الانتساب

### 2. **إدارة الدورات (Courses Management)**

- عرض جميع الدورات في جدول منظم
- البحث والتصفية حسب:
    - الحالة (مسودة/منشورة/مؤرشفة)
    - النوع (أون لاين/أوف لاين)
- عرض معلومات الدورة:
    - الفئة والمدرب
    - السعر والمستوى
    - عدد الطلاب المسجلين
- إمكانية إضافة وتعديل وحذف الدورات

### 3. **إدارة الطلاب (Students Management)**

- قائمة بجميع الطلاب المسجلين
- البحث والترتيب حسب:
    - الاسم
    - تاريخ التسجيل
- عرض إحصائيات الطلاب:
    - الطلاب النشطين
    - المتدرجين (أكملوا الدورة)
    - الدورات المسجلة لكل طالب
- متابعة تقدم الطالب

### 4. **إدارة المدربين (Instructors Management)**

- عرض جميع المدربين في تصميم بطاقات جذابة
- معلومات المدرب:
    - الاسم والبريد الإلكتروني والهاتف
    - التخصص وسنوات الخبرة
    - حالة النشاط
    - الدورات التي يديرها
- البحث والتصفية حسب الحالة
- إمكانية تعديل وحذف بيانات المدربين

### 5. **إدارة المراجعات والتقييمات (Reviews Management)**

- عرض جميع التقييمات والمراجعات
- الفلترة حسب الحالة:
    - قيد الانتظار (Pending)
    - موافق عليها (Approved)
    - مرفوضة (Rejected)
- عرض التقييمات:
    - عدد النجوم (1-5)
    - تعليقات الطالب
    - معلومات الدورة والطالب
- إمكانية الموافقة أو رفض التقييمات
- إحصائيات التقييمات:
    - إجمالي التقييمات
    - متوسط التقييم
    - عدد التقييمات المعلقة

### 6. **إعدادات النظام (Settings)**

- الإعدادات العامة:
    - اسم الموقع والوصف
    - البريد الإلكتروني والهاتف
    - العنوان
- إعدادات الشكل والألوان:
    - اللون الأساسي
    - اللون الثانوي
- إعدادات الميزات:
    - تفعيل/تعطيل التسجيل الجديد
    - تطلب تحقق البريد الإلكتروني
    - وضع الصيانة
- حدود النظام:
    - الحد الأقصى للطلاب في الدورة

## البنية الأساسية للقاعدة البيانات

### الجداول المنشأة:

1. **Users** (مع إضافة حقل role)
    - student (طالب)
    - instructor (مدرب)
    - admin (مسؤول)

2. **Categories** - الفئات التدريبية

3. **Courses** - الدورات
    - type: online/offline
    - status: draft/published/archived
    - level: beginner/intermediate/advanced

4. **Instructors** - المدربين والمحاضرين

5. **Enrollments** - التسجيل والالتحاق بالدورات

6. **Lessons** - الدروس والمحاضرات

7. **LessonCompletions** - تتبع إكمال الدروس

8. **Reviews** - التقييمات والمراجعات

## المسارات (Routes)

```
GET    /admin/dashboard              - لوحة البيانات الرئيسية
GET    /admin/courses                - إدارة الدورات
GET    /admin/courses/create         - إضافة دورة جديدة
GET    /admin/courses/{id}           - عرض تفاصيل الدورة
GET    /admin/courses/{id}/edit      - تعديل الدورة
GET    /admin/students               - إدارة الطلاب
GET    /admin/instructors            - إدارة المدربين
GET    /admin/reviews                - إدارة التقييمات
GET    /admin/settings               - الإعدادات
```

## الخطوات للبدء

### 1. تشغيل الهجرات:

```bash
php artisan migrate
```

### 2. إنشاء حساب إداري (اختياري):

```bash
php artisan tinker
# ثم في tinker:
User::create(['name' => 'Admin', 'email' => 'admin@almarf.com', 'password' => bcrypt('password'), 'role' => 'admin']);
```

### 3. الوصول إلى لوحة التحكم:

```
http://localhost:8000/admin/dashboard
```
